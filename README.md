
## Description

Project built with [Nest](https://github.com/nestjs/nest) framework TypeScript with [Keyclock](https://www.keycloak.org/) as Identity Provider.

## Setup Keycloak
```bash
$ docker-compose up
```
Access the instance, configure new realm with users and roles (To be detailed)


## Project setup

```bash
$ npm install
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Run tests

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
